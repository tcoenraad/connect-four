Requirements
------------
* [Haskell Platform](https://www.haskell.org/platform/)
* [Bower](http://bower.io/#install-bower)

How to set-up
-------------
```bash
    $ cabal install --only-dependencies
    $ bower install
```

How to run
----------

### Run app
```bash
	$ cabal run
```
* Web monitor on `http://localhost:3500`
* Server on `localhost 3501`

### Run tests
```bash
    $ cabal test
```
